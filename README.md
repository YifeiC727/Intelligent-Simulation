# JoyAI-Sim

京东具身智能仿真交互平台 - 前端交互原型

## 项目简介

JoyAI-Sim 是一个基于 Web 的具身智能仿真交互平台，用户通过键鼠操控机械臂完成物体抓取与归位任务，同时与 AI 对手进行同场竞技。

## 页面结构

| 页面 | 文件 | 说明 |
|------|------|------|
| 首页 | `landing.html` | 场景选择、关卡入口、世界探索进度 |
| 引导页 | `onboarding.html` | 关卡介绍、操控说明、进入游戏前的过渡 |
| 游戏页 | `game.html` | 核心玩法页面，含教程模式与正式关卡 |
| 入口 | `index.html` | 重定向至 landing.html |
| 排行榜组件 | `leaderboard.css` + `leaderboard.js` | 游戏页内嵌浮动排行榜（正式关卡显示） |

## 当前关卡

- **书桌场景 - 新手教学**：教学模式，不限时，1 个物体
- **茶几场景 - 玩具车**：初级关卡，12 分钟，1 个物体

## 本地运行

纯静态页面，无需构建工具。直接在浏览器中打开 `index.html` 即可，或使用任意静态服务器：

```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

## 技术栈

- 原生 HTML / CSS / JavaScript（单文件，无框架依赖）
- Google Fonts (Inter)
- Canvas 2D（粒子背景效果）

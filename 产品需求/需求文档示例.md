# 模型体验页 - 视觉 @吕若璠

## 能力1 - 子首页一键同款

用户可通过导航栏/二级首页/新对话三个 tab 进入此页面，页面展示可复用的 sample。

> 设计美感参考（首页）：对话框宽度按设计审美调整，case 展示页上下左右留白减少，提升冲击感

---

## 一、首页

左侧模型导航栏之外仅画布页（对话页），包含以下部分：

| 分类 | 需求描述 |
|------|----------|
| 我的资产 | 详见 1.3 节 |
| 页签选择 | 发现 / 创作 |
| search 框 | 详见 1.4 节 |
| sample 展示区 | 详见第二章「发现」 |
| 对话框 | 固定置底（悬浮/和语言类似占底均可） |

### 1.1 登录交互

| 场景 | 交互逻辑 |
|------|----------|
| 二级首页点击 | 点击视频图标，跳转显示本链接（用户已登录状态） |
| 直接 URL 跳转 | 直接 URL 跳转该链接 |
| 首次登录 | 用户进行任何操作时，验证是否处于内网环境，唤起 ERP 登录页，验证 ERP 后登录，显示该页 |
| 多次登录 | 验证是否处于内网环境，（跳过登录页）显示该页（用户已登录状态） |

### 1.2 页签选择

包含**发现**、**创作**两个页签，用户从导航栏/二级首页 tab 进入页面，未发生新对话前，默认「发现」页。

### 1.3 我的资产

| 功能 | 需求描述 |
|------|----------|
| 进入资产 | 点击进入我的资产，所有作品按照月份聚类呈现 |
| 查看详情 | 点击对应图片则跳转至对应图片详情，用户可点击下载或重新生成；若为图片编辑也将对应展示原始图片 |
| 重新生成 | 点击重新生成后回到对话页，提示词重新带入对话框，用户点击右下角箭头开始生成；图片编辑则也将同步带入原始图片 |

### 1.4 Search 框

| 功能 | 需求描述 |
|------|----------|
| 展开搜索 | 点击搜索按钮，展开搜索框 |
| 输入搜索 | 搜索框左侧为输入区域，输入内容按回车键后，定向搜索历史生成记录提示词中包含该内容的记录，集中展示 |
| 时间筛选 | 右侧为时间栏和类型栏目，点击时间栏下拉时间选择框，点击对应时间可定向筛选 |
| 类型筛选 | 点击类型栏下拉"全部/图片/视频"选择栏，点击后定向筛选 |
| 筛选提示 | 检索筛选后对话页底层出现灰色提示文字，标记筛选条件 |
| 重置条件 | 同时选中时间栏的"全部"、类型栏的"全部"状态且输入框内无内容则恢复原始全部历史记录 |

## 二、发现（sample 展示区）

> 展示内容由产品配置/批量上传

### 2.1 展示逻辑

| 规则 | 说明 |
|------|------|
| 内容混合 | 无论用户选择 video/image，sample 页唯一，图像 & 视频混合展示 |
| 视频展示 | sample 视频为静态展示，默认首帧；若为视频类型鼠标移动至 sample 后自动播放视频 |

### 2.2 交互逻辑 - 点击 case 进入全屏展示页

| 区域 | 说明 |
|------|------|
| 左侧 | 展示图像/视频，用底色边框拓展为统一尺寸 |
| 右侧 | 白底色黑字展示 prompt、用户选择的生成条件（模型、比例、清晰度）、复制键 |
| 关闭 | 页面关闭键 |
| 视频控件 | 暂停、进度条、下载键 |

### 2.3 交互逻辑 - 鼠标悬浮至 case

| 内容类型 | 悬浮框按钮 |
|----------|-----------|
| 视频 | "一键同款"、"下载" |
| 图像 | "一键同款"、"下载"、"编辑" |

### 2.4 功能说明

| 功能 | 说明 |
|------|------|
| 功能1 - 一键同款 | 用户点击一键同款后，将该 video/image 生成的全部输入、配置，自动在编辑框输入，用户点击发送键可开始生成 |
| 功能2 - 下载 | 用户点击下载图标后，文件自动下载至默认路径 |
| 功能3 - 编辑 | 点击编辑把图片带入输入框作为参考图 |

## 三、创作（历史对话区）

| 分类 | 需求描述 |
|------|----------|
| 入口 | 点击「创作」展示历史对话 |
| 左侧内容 | 生成的作品，所有任务默认仅生成 1 个 |
| 右侧内容 | 输入的 prompt 及选择的属性 |
| 布局比例 | 因默认生成单张图/单视频，比例采用 图/视频:prompt = 1:2，或者类似文本对话展示形式（右上文字，左下生成内容） |
| 底部提示 | 对话框拉到底，展示提示词"我也是有底线的" **P1** |

## 四、对话框

| 场景 | 说明 |
|------|------|
| 浏览状态 | 用户在发现页/对话页浏览 |
| 输入状态 | 用户输入，展开逻辑看 UI 设计效果确定是否需要 |

> 交互逻辑见下两部分

### 特别关注

| 优先级 | 需求描述 |
|--------|----------|
| ❗️ 新增关注 | 记录上一次用户选择的生成类型（图像/视频），默认选到上一次任务类型；用户首次登录/退出后再登录默认选择图像 |
| ❗️ 新增关注 | 再次登录后保留用户历史记录 |

## 五、新增需求 - 视觉创作新手首页

### 5.1 出现逻辑

| 用户类型 | 行为 |
|----------|------|
| 新用户 | 首次点击视觉创作功能（之前从未在视觉功能页进行交互并生成内容）的用户会出现此页，起新手创意引导作用 |
| 老用户 | 已在视觉创作功能对话页生成过内容的老用户将不会再看到此页面，直接进入对话页 |

### 5.2 页面布局

| 区域 | 内容描述 |
|------|----------|
| 上方 | 动态图标 + 大字 slogan |
| 中部 | 空白对话框（正常高度版本） |
| 下方 | 横向排版的五个创意提示词模版卡片（包含生图、图片编辑、生视频），均为静止状态（生视频模版卡片仅鼠标悬停时开始播放） |

### 5.3 点击逻辑

| 操作 | 说明 |
|------|------|
| 直接输入 | 可在对话框直接输入提示词/参考图，点按右下角变蓝箭头跳转至对话页等待结果 |
| 点击模版卡片 | 对应提示词自动复制进对话框（可在此基础上自行修改），点按右下角变蓝箭头跳转至对话页等待结果 |

### 5.4 模版提示词 - 卡片结构

| 元素 | 说明 |
|------|------|
| 卡片结构 | 竖向卡片，上方为效果图/视频，下方为模版标题 + 详细提示词 |
| 图片编辑卡片 | 上方效果图左下角有缩略版原图 |
| 悬停效果 | 鼠标悬停后会有突出效果，若为生视频模版卡片则悬停后视频自动播放 |

### 5.5 其他模块

| 模块 | 说明 |
|------|------|
| 对话框 | 同对话页交互框 |
| 资产 | 同前 |
| 搜索 | 同前 |

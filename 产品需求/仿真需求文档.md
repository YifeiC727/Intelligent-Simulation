# JoyAI-Sim 具身智能仿真平台 - 首页需求文档

## 能力概述

首页是平台的主入口，采用全屏场景概览图 + 四层世界热区的布局。用户可通过点击热区切换场景、查看关卡，或点击"立即挑战"快速进入书桌新手教学。

### 功能模块总览

| 模块 | 说明 |
|------|------|
| 全屏场景概览图 | 四层世界建筑（书桌/茶几/办公室/CBD），缩放90%居中偏左展示 |
| 场景标注 | 建筑左侧四级标注（01 DESK/02 TEA TABLE/03 OFFICE/04 CBD），带连接线+末端圆点指向对应层 |
| 四个热区 | 对应四层世界，hover显示打字机场景名+拖尾特效，点击切换任务面板 |
| 顶部导航栏 | 固定顶部，左侧 Logo，右侧「关于」按钮 |
| Hero 区域（CTA） | 左侧文案区，含标题、副标题、「立即挑战→」和「关于」按钮；CTA 顶部位置固定不随任务数变化 |
| 任务面板 | Hero 区域下方的悬浮玻璃面板，展示当前场景关卡列表，默认显示书桌场景，固定尺寸，超出可下滑 |
| 世界探索进度 | 右侧固定面板（垂直居中），圆环进度 + 章节时间轴 |
| 「关于」面板 | 右侧滑出抽屉，含平台介绍、技术标签、仿真抓取动图 |
| 热区拖尾特效 | 鼠标在热区内移动时产生星尘粒子拖尾 |

## 一、全屏场景概览图

### 1.1 背景图

- 使用 `assets/scenes/首页示意1.png` 作为全屏背景
- 图片缩放至 90%（`width:90%; height:90%; object-fit:contain`）
- 整体向左偏移 5%（`translateX(-5%)`），为右侧面板留出空间
- 纯黑背景底色

### 1.2 四个热区

基于参考区域图（`assets/scenes/首页示意1_参考区域.png`）定位，从上到下：

| 热区 | 位置（CSS） | data-scene |
|------|------------|------------|
| 书桌 | top:4%, left:44%, w:36%, h:14% | desk |
| 茶几 | top:22%, left:42%, w:40%, h:14% | table |
| 办公室 | top:42%, left:40%, w:44%, h:14% | office |
| CBD | top:65%, left:38%, w:48%, h:18% | cbd |

### 1.3 热区交互

- **Hover**：不显示边框/背景高亮，仅触发顶部打字机效果 + 鼠标拖尾特效
- **点击**：切换左下任务面板到对应场景
- **打字机效果**：页面顶部居中显示打字动画（如 `// DESK_SCENE`），离开热区后消失
- **拖尾特效**：鼠标在热区内移动时产生星尘粒子拖尾（与 V2 一致）

## 二、顶部导航栏 + CTA

### 2.1 导航栏

- 固定顶部通栏，延迟 0.3 秒淡入
- 左侧：Logo 图片（`assets/ui/JoyAI.png`）
- 右侧：「关于」按钮，半透明背景 + 细边框

### 2.2 Hero 区域

- 位于左侧（`left:64px`），容器纵向撑满视口（`top:0; bottom:0`），通过 `padding-top:18vh` 将文案区定位在固定高度，延迟 0.5 秒淡入
- **CTA 位置固定**：平台标签、主标题、副标题、按钮组均设为 `flex-shrink:0`，其顶部位置不随下方任务面板的关卡数量变化而上下移动
- **平台标签**："JOYAI-SIM 具身智能仿真平台"，紫色小字，大写
- **主标题**："操控机器人，探索**真实**世界。"，大号粗体渐变，"真实"紫色高亮
- **副标题**："从桌面开始，逐步探索更大的世界。同场竞技，看人类直觉能否胜过机器算法。"
- **按钮组**：
  - 「立即挑战 →」：紫色渐变实心填充（#6B2FF7→#8B5CF6），hover 上浮+发光
  - 「关于」：透明底 + 白色细边框，hover 加亮

## 三、任务面板（Hero 区域下方）

### 3.1 面板布局

- 位于 Hero 文案区下方（`margin-top:24px`），随 Hero 容器定位于页面左侧
- **固定尺寸**：width:345px, height:300px（不随场景关卡数量变化，保持各场景一致）
- 延迟 0.5 秒随 Hero 一同淡入
- **悬浮玻璃面板风格**：深黑蓝半透明 `rgba(8,10,20,0.75)` + `blur(20px)` + 紫色细边框 `rgba(140,120,255,0.14)` + 投影 `0 8px 32px`

### 3.2 面板内容

- **标题**：当前场景名（如"书桌场景"），大写小字 + 左侧紫蓝渐变竖条装饰
- **关卡列表**：垂直排列，可下滑，每张卡片含图标、名称、描述、状态徽章

### 3.3 卡片形态与图标

- 每张卡片：物品名做主标题，关卡等级（初级/中级/进阶关卡，或教学模式说明）做副标题小字，右侧状态徽章
- **多形状图标**：不同关卡使用不同 SVG 图标形状，避免单一六边形，共 8 种形状配 5 种颜色主题：

| 图标 | 形状 | 颜色 |
|------|------|------|
| compass | 指南针 | 青 cyan |
| cube | 立方体 | 紫 purple |
| flask | 烧瓶 | 琥珀 amber |
| target | 靶心 | 青 cyan |
| diamond | 钻石 | 玫红 rose |
| star | 星形 | 琥珀 amber |
| hexagon | 六边形 | 紫 purple |
| globe | 地球 | 绿 green |

- **状态徽章**：
  - 「开始」（绿色）：可进入的关卡，所有可玩关卡统一使用，保持平行
  - 「即将开放」（暗灰）：未解锁关卡，卡片半透明且不可点击

### 3.4 场景数据

默认显示书桌场景：

| 场景 | 物品名 | 副标题 | 状态 |
|------|--------|--------|------|
| 书桌 | 新手教学 | 教学模式 · 不限时 · 1个物体 | 开始（可进入） |
| 书桌 | 文具盒 | 初级关卡 | 即将开放 |
| 书桌 | 涂改液 | 中级关卡 | 即将开放 |
| 茶几 | 玩具车 | 初级关卡 | 开始（可进入） |
| 茶几 | 纸巾盒 | 中级关卡 | 即将开放 |
| 办公室 | 即将开放 | 敬请期待 | 即将开放 |
| CBD | 即将开放 | 敬请期待 | 即将开放 |

> 茶几场景卡片格式与书桌场景一致；茶几「玩具车·初级」为可进入关卡。

### 3.5 交互逻辑

- **热区点击**：切换面板到对应场景的关卡列表
- **"立即挑战"按钮**：显示书桌场景 + 高亮第一张可进入卡片（脉冲发光动画）
- **点击可进入的关卡**：跳转至 onboarding.html，携带关卡参数
- **卡片 hover**：向右轻移（`translateX(3px)`）+ 紫色高亮边框
- **下滑逻辑**：关卡数量超出面板固定高度时，列表区域可垂直滚动；
  - 滚动条紫色，宽 5px（Firefox 使用 `scrollbar-width: thin`）
  - 内容溢出时列表底部显示渐隐遮罩，提示"下方还有更多"；滑到底部时遮罩消失
  - 通过 `updateScrollHint()` 在渲染与滚动时动态切换 `scrollable` / `at-bottom` 状态类

### 3.6 智能逻辑（后续迭代）

| 用户状态 | 行为 |
|---------|------|
| 从未玩过 | 展示书桌场景，只有新手教学解锁 |
| 某关未通 | 展示该场景的关卡 |
| 已通关 | 展示下一个场景的关卡 |

状态通过 localStorage 持久化（后续实现）。

## 四、右侧信息栏

### 4.1 布局

- 固定定位，垂直居中（`top:50%; right:24px; transform:translateY(-50%)`），width:280px
- 仅世界探索进度一张卡片
- 延迟 1.2 秒淡入

### 4.2 世界探索进度卡片

**卡片样式**：
- `rgba(10,13,24,0.82)` 背景 + `blur(18px)` + 紫色细边框 `rgba(145,130,255,0.16)`
- Hover 时边框加深至 `rgba(143,95,255,0.30)`

**圆环进度**：
- 直径 112px，环宽 8px
- 底色 `rgba(255,255,255,0.07)`，填充紫蓝渐变 `#7A4DFF → #A45BFF`
- 中心：上方"探索进度"11px + 下方百分比 30px

**章节时间轴**：
- 左侧 1px 垂直连接线
- 节点状态：当前（紫色发光 #8C52FF）、已解锁（紫边空心）、锁定（暗灰边）
- 每章节含：Chapter编号(10px)、标题(13px)、进度条(4px紫渐变)、完成数量

**章节数据**：

| Chapter | 标题 | 任务数 | 状态 |
|---------|------|--------|------|
| 01 | Desk Exploration | 0/3 | active |
| 02 | Tea Table Exploration | 0/2 | unlocked |
| 03 | Office Tasks | 0/??? | locked |
| 04 | CBD Exploration | 0/??? | locked |

## 五、"关于"面板

### 5.1 入口与交互

- 点击顶部或 Hero 区域的「关于」按钮触发
- 全屏半透明黑色遮罩 + 毛玻璃模糊
- 面板从右侧滑入（400px宽），深色半透明 + 强模糊
- 点击遮罩或 × 按钮关闭

### 5.2 内容结构

- **标签**："ABOUT"，蓝色小字
- **标题**："京东具身智能 仿真交互平台"
- **描述**：平台技术能力与资产规模介绍
- **技术标签组**：Gaussian Splatting / Physics Engine / JoyBuilder / WebGL2 / 53K+ 3D Assets
- **演示动图**：仿真抓取效果

## 六、入场动画节奏

| 顺序 | 模块 | 延迟 |
|------|------|------|
| 1 | 背景场景图 | 立即显示 |
| 2 | 顶部导航栏 | 0.3s |

## 七、响应式适配

- 移动端（≤ 768px）：标题字号缩小，文案区域收窄
- 移动端隐藏：任务面板、右侧信息栏

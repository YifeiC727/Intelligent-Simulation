# JoyAI-Sim 具身智能仿真平台 - 首页需求文档

## 能力概述

首页是平台的主入口，用户打开后首先看到 3D 粒子工作台动画，随后进入主界面进行关卡选择或了解平台信息。

### 功能模块总览

| 模块 | 说明 |
|------|------|
| 3D 工作台 + 粒子效果 | 全屏 canvas 背景，加载桌面模型，50 万粒子表面采样，点击后聚合成 3D 桌面工作台形态 |
| 顶部导航栏 | 固定顶部通栏，左侧 Logo "JoyAI"，右侧「关于」按钮 |
| Hero 区域（CTA） | 左上文案区，含标题、副标题、平台标签 |
| 关卡选择面板 | 左侧卡片列表，按场景分组展示关卡，可滚动，点击进入关卡 |
| "关于"面板 | 右侧滑出抽屉，含平台介绍、技术标签、仿真抓取动图演示 |
| 侧边栏 | 左侧隐藏导航，鼠标靠近左边缘触发滑出，含 AI Models 导航 |
| 点击聚合提示 | 居中覆盖层，"CLICK TO CONVERGE" 引导用户触发粒子聚合 |

## 一、3D 工作台 + 粒子效果

### 1.1 整体效果

- 全屏纯黑背景，加载 `desk-split.glb` 桌面模型（含 65 个部件）
- 50 万粒子均匀覆盖模型表面，带柔和辉光后处理，整体画面呈现深邃太空感
- 模型加载后自动缩放居中，正面朝向用户
- 粒子从远处看呈星云状轮廓，聚合后清晰还原出桌面工作台的完整形态

### 1.2 粒子系统

- **初始状态**：粒子在模型表面附近薄雾般随机散布，持续缓慢漂浮呼吸，整体轮廓隐约可辨
- **目标状态**：粒子聚合还原为完整桌面模型形态，表面细节清晰
- **粒子大小**：大小不一（小粒子为主 + 少量大亮点），形成星尘质感，带微弱呼吸缩放
- **着色方案**：基于部件位置分区着色，整体色调偏冷，局部暖色点缀
  - 桌腿：深冷色近乎纯黑，微弱蓝色
  - 桌面：深棕暖底，随机分布琥珀色光斑
  - 桌上物体：按左中右区域渐变——左侧电光蓝、中间深靛紫、右侧热琥珀橙
  - 机械臂：冷蓝-紫色调，顶端偶现蓝白高光
  - 前方粒子整体偏暖，后方偏冷，增加空间深度感

### 1.3 点击聚合动画

- 页面中央显示 "CLICK TO CONVERGE" 引导按钮：圆形边框 + 播放图标，带呼吸脉冲缩放动画（边框颜色在透明与蓝色间交替），蓝色微光晕
- 用户点击后，引导层 1 秒淡出消失
- 粒子在约 6 秒内平滑聚合成模型形态，节奏先慢后快（前半段缓慢启动，后半段快速收敛）
- 聚合接近完成（约 80%）后，鼠标靠近粒子可产生轻微涡旋偏移交互（粒子绕鼠标旋转而非被推开，带上下波动）
- 关卡选择面板始终在聚合提示层之上，不被遮挡

## 二、顶部导航栏 + CTA

### 2.1 导航栏

- 固定顶部通栏，页面加载后延迟淡入（上移渐显）
- 左侧：Logo "Joy**AI**"，其中 "AI" 为品牌蓝色高亮，白色粗体
- 右侧：「关于」按钮，半透明背景 + 细边框，hover 时背景加深

### 2.2 Hero 区域

- 位于左上角，导航栏下方，延迟淡入（晚于导航栏）
- **平台标签**："JOYAI-SIM 具身智能仿真平台"，紫色小字，大写字母，宽字距
- **主标题**："操控机械臂，挑战 AI 对手。"，大号粗体，四色渐变（白→紫→蓝→白），视觉焦点
- **副标题**："进入具身仿真世界，完成抓取任务，同场竞技，看人类直觉能否胜过机器算法。"，低对比度浅灰色，行距宽松

## 三、关卡选择面板

### 3.1 面板布局

- 位于页面左侧偏下方，固定定位，延迟淡入（最晚出现，晚于 Hero 区域）
- 超出可视区域时支持滚动，自定义细滚动条
- 标题："关卡选择"，蓝色粗体

### 3.2 关卡卡片

- 垂直列表排列，卡片间距紧凑
- 卡片风格：圆角，深色半透明毛玻璃背景，细边框（几乎不可见）
- 每张卡片包含：关卡名称（白色粗体）、难度星级（金色填充/灰色空心）、状态徽章（绿底"已通关"）、元信息（限时·物体数，灰色小字）
- Hover 态：卡片右移 + 边框变蓝 + 内部出现蓝紫渐变叠层 + 柔和蓝色外阴影
- 锁定态：整体半透明化，不可点击，右侧显示锁图标，标注"即将开放"

### 3.4 跳转逻辑

- 点击可进入的关卡卡片后跳转至加载等待页（加载仿真环境期间展示关卡信息）
- 携带参数：关卡 ID、中文名、物体数、限时秒数等

### 3.5 关卡状态逻辑

| 条件 | 效果 |
|------|------|
| 用户进入过新手教学关卡 | 解锁其他所有可进入关卡 |
| 用户成功完成新手教学任务（物体放置成功） | 新手教学卡片显示"已通关"徽章 |

状态通过 localStorage 持久化。

## 四、"关于"面板

### 4.1 入口与交互

- 点击顶部「关于」按钮，打开右侧抽屉面板
- 背景出现全屏半透明黑色遮罩，带毛玻璃模糊效果
- 面板从右侧滑入，深色半透明背景 + 强模糊 + 左侧细边框，整体质感与卡片风格统一
- 点击遮罩或右上角 × 按钮或按 Esc 关闭面板，× 按钮 hover 时高亮

### 4.2 内容结构

- **标签**："ABOUT"，蓝色小字，大写宽字距
- **标题**："京东具身智能 仿真交互平台"，白色大号粗体
- **描述**：介绍平台技术能力与资产规模，低对比度灰色正文，行距宽松
- **技术标签组**：Gaussian Splatting / Physics Engine / JoyBuilder / WebGL2 / 53K+ 3D Assets，标签为深色小圆角胶囊，细边框，灰色文字
- **演示动图**：圆角裁切，细蓝色边框，展示仿真抓取效果

## 五、侧边栏

- 左边缘常驻显示一条白色细竖条，右侧附带箭头，箭头做水平方向的脉冲呼吸动画，提示用户可展开
- 鼠标靠近左边缘隐形热区时，白色背景侧边栏从左侧滑入展开
- 顶部品牌：Logo "Joy**AI**-Sim"，AI 为蓝色高亮
- 导航项列表，hover 时浅灰背景高亮：
  - Home（默认选中，蓝色高亮态）
  - 分组标签 "AI MODELS"（灰色大写小字）
  - 视觉理解、视觉生成、具身交互
- 鼠标离开后延迟约 0.4 秒自动收起，侧边栏打开时白色竖条隐藏

## 六、入场动画节奏

页面各模块依次淡入，营造层次感：

1. 3D 粒子背景（立即渲染）+ 点击聚合提示层
2. 顶部导航栏（延迟约 0.3 秒淡入）
3. Hero 文案区域（延迟约 0.5 秒淡入）
4. 右上角操控提示（延迟约 1 秒淡入）
5. 关卡选择面板（延迟约 1.2 秒淡入）
6. 底部滚动提示（延迟约 1.5 秒淡入）

所有淡入动画均为「上移 + 渐显」效果。

## 七、响应式适配

- 移动端（≤ 768px）：标题字号缩小，文案区域收窄
- 移动端隐藏：控制面板、侧边栏、关卡选择面板

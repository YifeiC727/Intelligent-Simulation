# JoyAI-Sim 具身智能仿真平台 - 游戏页需求文档（新手教学关）

## 能力概述

游戏页（game.html）是核心仿真交互页面。本文档仅描述新手教学关卡（level=tutorial）的需求。用户从引导页跳转进入，通过键盘操控机械臂完成物体抓取与放置任务，包含完整的分步引导流程。

---

## 一、页面整体布局

新手教学关为全屏单画面（无 AI 分屏），上中下三栏布局，全屏不可滚动：

- **顶栏（Header）**：Logo、关卡名称（"新手教学"）、正计时器、物体完成进度点、暂停按钮
- **中间主区域**：全屏仿真操作画面
- **底栏（Footer）**：按键快捷指南（教学完成后显示）

### 1.1 顶栏元素说明

- **Logo**：左上角显示 JoyAI-Sim 品牌标识
- **关卡名称**：Logo 右侧显示当前关卡名（"新手教学"）
- **正计时器**：居中显示，从 0:00 开始累加
- **完成进度点**：每个物体对应一个圆点，放入目标位置后点亮（白→蓝色发光）
- **暂停按钮**：右上角，点击或按 Esc 可暂停游戏

新手教学关隐藏的 UI 元素：分数对比区、连击倍率、排行榜。

## 二、键鼠操控映射

| 按键 | 操作 | 说明 |
|------|------|------|
| Tab | 切换控制左臂 / 右臂 | 一次只控制一侧 |
| W / S | 末端前进 / 后退 | 沿基座 X 轴平移 |
| A / D | 末端左移 / 右移 | 沿基座 Y 轴平移 |
| ↑ / ↓ | 末端上升 / 下降 | 沿基座 Z 轴平移 |
| ← / → | 末端俯仰旋转 | 绕局部 Y 轴旋转，每帧 ±1° |
| 空格 | 夹爪抓取 / 释放 | 吸附式抓取，按一次抓取，再按一次释放 |
| Q | 旋转头部视角向左 | 调整俯瞰角度 |
| E | 旋转头部视角向右 | 调整俯瞰角度 |
| H | 复位 | 夹爪和机械臂回到初始位姿 |
| Esc | 暂停 / 继续 | 切换暂停状态 |

## 三、新手教学流程

### 3.1 教学流程总览

新手教学分三个阶段依次进行：

```
阶段一：聚光灯介绍（Spotlight Intro）
    ↓ 点击推进
阶段二：按键交互教学（Key Tutorial）
    ↓ 全部按键学完 / 点击跳过
阶段三：UI 元素聚光灯（Post-Key Spotlight）
    ↓ 点击推进
进入正式教学游玩（Step Guide + Tutorial Overview）
```

### 3.2 阶段一：聚光灯介绍

使用全屏暗色遮罩 + 高亮区域切口 + 白色提示气泡的方式，逐步介绍界面。每步点击任意处推进到下一步。

**聚光灯期间键盘输入被锁定**，用户无法操控机械臂，必须点击推进。

| 步骤 | 高亮区域 | 提示文字 |
|------|---------|---------|
| 1 | 游戏主区域 | "欢迎来到新手教学！" |
| 2 | 按键教学区域 | "这里先帮你熟悉一下仿真游戏中的键盘使用" |
| 3 | 按键教学区域 | "先尝试按住 W 键让抓夹向前移动吧！" |

第 3 步点击后，进入阶段二的按键交互教学。

**注意**：按键教学期间，用户按下移动按键时，仿真环境中的机械臂会同步响应移动，提供实时视觉反馈。

### 3.3 阶段二：按键交互教学

顶部居中显示仿 Mac 键盘风格的按键卡片。用户需逐一按住对应按键达到要求时长，按键上会显示填充进度条，完成后打勾并自动进入下一组。

教学步骤：

| 步骤 | 按键 | 提示文字 | 要求 |
|------|------|---------|------|
| 1 | W → S | "按住 W 前进" → "按住 S 后退" | 各按住 1.2 秒 |
| 2 | A → D | "按住 A 左移" → "按住 D 右移" | 各按住 1.2 秒 |
| 3 | ↑ → ↓ | "按住 ↑ 上升" → "按住 ↓ 下降" | 各按住 1.2 秒 |
| 4 | ← → → | "按住 ← 向左旋转" → "按住 → 向右旋转" | 各按住 1.2 秒 |
| 5 | Q → E | "按住 Q 视角左转" → "按住 E 视角右转" | 各按住 1.2 秒 |
| 6 | 空格 | "按一下空格张开 / 关闭抓夹" | 按一下即可 |
| 7 | Tab | "按一下 Tab 切换左右臂" | 按一下即可 |
| 8 | H | "按一下 H 回到初始位姿" | 按一下即可 |

**交互规则：**
- 含多个子按键的步骤（如 W/S），需依次完成每个子按键
- 当前需要按的按键高亮显示，其余按键半透明
- 按住过程中按键上方出现灰色填充进度条
- 中途松手，进度条归零，需重新按住
- 用户按了非当前要求的键，忽略不响应
- 用户同时按多个键，仅响应当前要求的键，其余忽略
- 完成的按键显示绿色 ✓
- 全部步骤完成后显示 "操作学会了！" 提示，1.5 秒后自动进入阶段三
- 底部有「跳过」按钮，点击直接跳过按键教学

### 3.4 阶段三：UI 元素聚光灯

按键教学完成后，底部 Footer 变为可见，开始介绍 UI 元素：

| 步骤 | 高亮区域 | 提示文字 |
|------|---------|---------|
| 1 | 底部按键指南区 | "这里是按键指南，之后随时可以查看键位" |
| 2 | 计时器 | "这里显示游戏时间。新手关不计时，其他关卡会有倒计时" |
| 3 | 完成进度点 | "每个圆点代表一个物体，放入对应位置后会被点亮" |
| 4 | 暂停按钮 | "点这里可以暂停游戏" |
| 5 | 无（居中） | "现在，跟随分步教学，将高亮物体放进白色虚线框里吧！" |

### 3.5 教学游玩阶段

UI 元素介绍完成后，进入实际教学游玩。新手关只有 1 个物体，全程有分步引导。

**UI 面板：**

- **左侧步骤面板**（毛玻璃卡片）：标题 "新手引导"，显示当前进度（如 "1 / 4"），四步列表，当前步骤蓝色高亮，已完成步骤绿色打勾
- **顶部任务卡片**（居中圆角毛玻璃条）：显示当前步骤编号、标题、操作说明、相关按键图标，四个圆点指示器标记当前步骤

**分步引导**（由仿真环境实时检测触发）：

| 步骤 | 提示内容 | 显示按键 | 视觉效果 | 进入下一步的条件 |
|------|---------|---------|---------|----------------|
| 1 靠近目标 | "移动夹爪到 [物体名] 上方" | W A S D | 目标物体周围蓝色呼吸光环 | 夹爪进入物体 5cm 范围 |
| 2 抓取物体 | "按空格抓取！" | 空格 | 无 | 夹爪闭合且物体被抬起 |
| 3 搬运到目标区 | "移到虚线框内" | W A S D | 无 | 物体进入目标区正上方范围 |
| 4 放置物体 | "按空格释放物体" | 空格 | 夹爪锁定在目标位置，等待用户按空格 | 物体放入目标位置 |

**锁定状态说明**：仅步骤 4 中夹爪锁定，用户不按空格则无限等待。

**失败处理**：抓取失败或搬运中物体掉落时，物体回落桌面，提示"没关系，再试一次！"（显示 3 秒后淡出），重新回到步骤 1 的分步引导。

放置成功后显示 "完成了！"，随后直接进入结果页。

## 四、时间系统

- **正计时**：从 0:00 开始向上累加，显示格式为 M:SS
- **10 分钟提示**：到达 10 分钟时，页面右下角（Footer 上方）出现「看看结果？」按钮，不可关闭，持续悬停显示直到用户点击
- 点击按钮跳转至结果页
- **计时启动条件**：计时器不会在页面加载时立即开始，而是在玩家首次进行操作（任意鼠标/键盘交互）后才开始计时

---

## 五、暂停系统

- **触发**：点击顶栏暂停按钮，或按 Esc 键
- **效果**：全屏暗色遮罩 + 毛玻璃模糊，居中显示 "暂停" 标题
- **暂停期间**：计时停止，游戏逻辑冻结
- **选项**：
  - 「继续游戏」按钮（蓝紫渐变主按钮）
  - 「退出」按钮（跳转回首页 landing.html）
- **快捷键提示**：底部显示 "按 Esc 继续"
- **恢复**：点击「继续游戏」或再次按 Esc
- **限制**：玩家未开始操作时暂停不可触发；新手引导流程进行中（阶段一~三）暂停不可触发；阶段三介绍完暂停按钮后，教学游玩阶段可正常暂停

## 六、结果页

### 6.1 触发条件

- 玩家完成全部物体放置
- 玩家手动点击「看看结果？」按钮（10 分钟后出现）

### 6.2 结果页内容

全屏暗色遮罩 + 强模糊背景，居中展示：

**标题**（根据完成情况动态变化）：
- 物体放置成功 → "完成了！"
- 10 分钟到达用户手动结束（物体未完成） → "就差一点！"

**副标题**：显示用时和完成数（如 "用时 2:30，完成 1/1 个物体归位"）

**详细信息**：
| 维度 | 显示 |
|------|------|
| 完成数 | X/N |
| 用时 | M:SS |

**操作按钮**：
- 「进入下一关」（蓝紫渐变主按钮，跳转回首页 landing.html）
- 「返回首页」（半透明次按钮，跳转 landing.html）

## 七、操控面板（Controls FAB）

按键教学完成后，玩家画面左下角出现一个小型浮动按钮 "Controls"（带键盘图标）：

- 点击展开操控说明面板（白色毛玻璃卡片），列出全部键位
- 再次点击或点击 × 按钮收起
- 方便玩家随时查看键位

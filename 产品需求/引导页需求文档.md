# JoyAI-Sim 具身智能仿真平台 - 引导页需求文档

## 能力概述

引导页（onboarding.html）是用户从首页选择关卡后、进入仿真游戏前的过渡页面。通过 4 个阶段的信息展示，帮助用户了解关卡信息、预览场景、学习操控方式，并激发挑战欲望。

---

## 一、页面整体结构

- 全屏黑色背景，带低密度粒子漂浮动效
- 顶部导航栏：左侧 Logo "JoyAI"，右侧「跳过」按钮（点击直接进入游戏）
- 中央区域：分 4 个阶段依次展示内容，带淡入上移动画
- 底部：阶段指示圆点 + 整体进度条
- 支持手动点击「下一步」跳转阶段

---

## 二、四阶段引导流程

页面自动轮播 4 个阶段，每阶段停留约 15 秒，共约 60 秒后自动进入游戏。

### 阶段一：关卡简报

- 显示当前关卡名称（如"书桌 · 初级"）
- 标签组展示关卡关键参数：
  - 教学模式关卡：教学模式标签（高亮）、不限时、物体数量
  - 普通关卡：限时分钟数、物体数量、位置随机；若有分值随机或障碍物则额外高亮显示
- 一句话关卡描述

### 阶段二：场景预览

- 标题："即将进入挑战"
- 描述："桌面上 X 个物体等你归位"（X 为实际物体数）
- 仿真抓取动图预览

### 阶段三：操控方式

以键盘按键图标 + 文字说明的形式，展示操控指令：

| 按键 | 操作 |
|------|------|
| Tab | 切换控制左臂 / 右臂（一次只控一侧） |
| W / S | 末端前进 / 后退（沿基座 X 轴平移） |
| A / D | 末端左移 / 右移（沿基座 Y 轴平移） |
| ↑ / ↓ | 末端上升 / 下降（沿基座 Z 轴平移） |
| ← / → | 末端俯仰旋转（绕局部 Y 轴，每帧 ±1°） |
| 空格 | 夹爪张开 / 闭合（吸附式抓取） |
| Q / E | 旋转头部视角（左 / 右） |
| H | 夹爪和机械臂回到初始位姿 |
| Esc | 暂停 |

### 阶段四：激发挑战

- 标题："你能比机器人做得更好吗？"
- 展示 AI 机器人完成同一任务的用时（如"机器人完成同样任务用了 10 分 14 秒"），激发用户竞技欲望

---

## 三、导航与进度

- **跳过按钮**：右上角，点击直接跳转至游戏页面
- **下一步按钮**：中央下方，手动切换到下一阶段；在最后一个阶段点击则直接进入游戏
- **阶段指示点**：底部 4 个圆点，当前阶段高亮展开为条状
- **进度条**：底部细条，渐变色填充，反映整体倒计时进度

---

## 四、页面跳转逻辑

- **来源**：从首页关卡卡片点击进入，URL 携带关卡参数（关卡 ID、中文名、物体数、限时秒数、是否分值随机、是否有障碍物）
- **去向**：进度走完或用户手动跳过后，跳转至游戏页面（game.html），透传全部 URL 参数

---

## 五、响应式适配

- 移动端（≤ 768px）：标题字号缩小，操控方式改为单列排列
